thats also works. when user gets offline. In this is dummy `E-commerce` website.

# Tests

Some tests are written.

# How to get Started

Well clone this repo and run

```npm

npm install

```

To run the App run

```npm

npm start

```

## To test App

Run

```npm

npm run test

```

## To build this App for production

```npm

npm run build

```
